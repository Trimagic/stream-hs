# Stream HS

Local media preparation and streaming app.

## Run

Install dependencies:

```bash
npm run install:all
```

Start backend and client together:

```bash
npm run dev
```

Pass a source folder from the console:

```bash
npm run dev -- "C:\Videos"
```

or:

```bash
npm run dev -- --source "C:\Videos"
```

Open:

```text
http://localhost:3000
```

Defaults:

```text
Client:  http://localhost:3000
Backend: http://localhost:4000
Media:   ./media
```
